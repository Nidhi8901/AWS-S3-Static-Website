# AWS-S3-Static-Website

# Static Website Deployment on AWS S3

## Overview
This project demonstrates how to deploy a static website using AWS S3 static website hosting.

## Services Used
- AWS S3

## Steps
1. Created an S3 bucket
2. Uploaded HTML files
3. Enabled static website hosting
4. Configured bucket policy for public access
5. Verified deployment via browser
